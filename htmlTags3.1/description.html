HTML tag
- marcaj care ajuta browser-ul sa afiseze corect continutul paginii web

- Attributes contain extra information about the element that you don't want to appear in the actual content.
HTML comment

HTML object element

HTML special character
- caracterele speciale sunt specificate in meta data
